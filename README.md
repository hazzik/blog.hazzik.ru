My personal blog

How to use
==========

1. Clone the repository

  > $ git clone https://github.com/hazzik/blog.hazzik.ru.git

  > $ cd blog.hazzik.ru

2. Install dependencies

  > $ bundle install

3. Run

  > $ bundle exec octopress serve

4. The blog is now avaliable at [http://localhost:4000/](http://localhost:4000/)
